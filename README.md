# nvim setup

## things to get started
1. Install the Python3 bindings
  * brew install python3
  * pip3 install neovim
2. Install the Python2 bindings
  * pip2 install neovim
3. Install dependencies
  * brew install fzf
  * brew install rg
4. launch vim
  * a script in the init should set up the plugins automagically
